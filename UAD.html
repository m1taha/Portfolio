Uniform Axial Deformation

Background

This project was designed to calculate and determine the absolute shear stress acting along the steel cylinder which consisted of 3 cross-sections and calculating deformation displacement using Python Programming

Execution

The program required the user to enter outer, inner diameter of the 1st cross-section and diameter of the 2nd and 3rd cross-section, characteristic lengths, and relative shear stresses. The program uses arithmetic which were derived by performing stress analysis procedure of a rod.
